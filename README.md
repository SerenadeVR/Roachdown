# Roachdown

A casual VR shooter where you defend an abandoned kitchen from cockroaches. Two pistols, shake to reload, endless waves that get harder over time. How long can you survive?

Built for Meta Quest 2 and 3 by [SerenadeVR](https://github.com/SerenadeVR).
